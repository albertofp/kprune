## Dependencies

- [gum](https://github.com/charmbracelet/gum)
- [kubectl](https://github.com/kubernetes/kubectl)

## Installation

```shell
chmod +x kprune.sh
sudo mv kprune.sh /usr/local/bin/kprune
```

## Usage

1. Authenticate into kubernetes cluster
2. Be in the namespace you want to prune
3. `kprune`
